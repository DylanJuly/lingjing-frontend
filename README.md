# 灵境 - 多模态AI命理分析与心理抚慰引擎

## 项目简介

灵境是一个融合传统玄学与现代AI技术的健康生活助手应用，通过计算机视觉、大语言模型和智能推荐算法，为用户提供个性化的命理分析、健康管理和饮食建议。

## 核心功能

### 1. 多模态AI命理分析
- 面相分析：通过上传照片进行面部特征点提取与纹路分析
- 手相分析：掌纹识别与特征分析
- 八字推演：基于出生时空信息的深度推演
- 综合运势：结合多维度数据生成运势解析与心理引导

### 2. 中西医结合的动态健康画像库
- 实时健康数据：心率、睡眠、步数、血氧等指标监测
- 智能设备接入：支持智能手表等穿戴设备数据同步
- 自主填报：舌苔拍照识别、日常状态问诊
- 体质分析：中医体质辨识与西医生理指标综合评价
- 历史趋势：健康数据的历史变迁可视化

### 3. 基于"药食同源"的智能饮食推荐
- 个性化食谱：根据体质、运势、季节自动生成每日食谱
- 药食同源匹配：AI自动匹配食材组合与烹饪方法
- 营养成分分析：详细的营养成分与热量信息
- 智能排雷：根据过敏源和禁忌自动过滤不适宜方案

### 4. 生活节律与玄学时令的智能规划看板
- 灵境看板：结合传统历法与健康科学的每日行事准则
- 节气提醒：二十四节气与干支历的智能提醒
- 运势概览：事业、健康、财运、感情等多维度运势展示
- 今日建议：基于运势和健康状态的个性化建议

## 技术栈

- **框架**: Flutter 3.0+
- **状态管理**: Provider
- **网络请求**: Dio
- **本地存储**: SharedPreferences
- **图片处理**: Image Picker, Cached Network Image
- **图表展示**: FL Chart
- **路由管理**: GoRouter
- **日期处理**: Intl

## 运行项目

### 前置要求
1. 安装 Flutter SDK (>=3.0.0)
2. 配置 Android Studio / VS Code 开发环境
3. 连接 Android 设备或启动模拟器（也可在桌面运行）

### 安装依赖
```bash
flutter pub get
```

### 运行应用

#### 在 Android 设备/模拟器运行
```bash
flutter run
```

#### 在桌面（Windows/macOS/Linux）运行
```bash
flutter run -d windows  # Windows
flutter run -d macos    # macOS
flutter run -d linux    # Linux
```

#### 在 Web 浏览器运行
```bash
flutter run -d chrome
```

### 查看可用设备
```bash
flutter devices
```
